Data Structure

All data for this project was obtained from the ASO Mission in the form of raster files in GeoTIFF format.
We use four raw "snowoff" rasters, which we assume are static over time. This set consists of a digital elevation map (DEM) and maps of vegetation height, canopy density (percent of a pixel covered by canopy), and land surface class (rock/soil, vegetation, water, or glacier).
